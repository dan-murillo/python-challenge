# Module 3 Challenge: python-challenge

```python-challenge``` is a Python library, which contains two folders:

## 'PyBank' Folder

1. The 'Resources' folder contains the CSV file with a financial dataset.
2. The 'analysis' folder contains the results of the analysis run on Python.
3. The 'main.py' file contains the main script that was created, optimised, commented, and run to analyse the CSV file with the financial dataset.

## 'PyPoll' Folder

1. The 'Resources' folder contains the CSV file with all the data of the ballots of an election.
2. The 'analysis' folder contains the results of the analysis run on Python.
3. The 'main.py' file contains the main script that was created, optimised, commented, and run to analyse the CSV file with the election data.
